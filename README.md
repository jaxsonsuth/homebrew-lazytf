# homebrew-lazytf

Homebrew tap for `lazytf`.

## Install

```bash
brew tap jaxsonsuth/lazytf
brew install lazytf
```

## Notes

- This formula currently builds `lazytf` from source.
- The `lazytf` release workflow can update this formula automatically when `HOMEBREW_TAP_TOKEN` is configured in `jaxsonsuth/lazytf`.
